SOBRE

Bem-vindo ao meu mundo digital! Sou desenvolvedora web fullstack com experiência em uma variedade de tecnologias, trabalhando para transformar ideias criativas em experiências digitais interativas e funcionais.

Com conhecimentos sólidos em HTML, CSS, JavaScript, jQuery, Bootstrap e Node.js, estou comprometida em entregar soluções web eficazes e visualmente impressionantes. Minha abordagem abrangente abrange desde a construção de interfaces de usuário intuitivas até o desenvolvimento de back-ends com Node.js e a integração de APIs para criar experiências web dinâmicas e escaláveis.

Ao longo da minha jornada, tenho colaborado em uma variedade de projetos desafiadores, onde minha paixão pelo desenvolvimento web me impulsiona a buscar constantemente novos conhecimentos e aprimorar minhas habilidades. Estou sempre ansiosa para enfrentar novos desafios e contribuir para o sucesso de projetos inovadores.
